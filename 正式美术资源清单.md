## 十一、资源命名规范总结

```
Resources/Sprites/
├── Towers/     tower_{name}_lv{1-3}.png      # name: archer/mage/ice/cannon/poison/goldmine
├── Enemies/    enemy_{name}.png              # name: infantry/assassin/knight/flyer/healer/slime/rogue/shieldmage/boss_dragon/boss_giant
│               enemy_{name}_walk.png         # 行走SpriteSheet（一行水平排列）
├── Effects/    projectile_{name}.png         # name: arrow/magic_bolt/ice_shard/cannonball/poison_glob/ice_beam
├── Maps/       tile_{name}.png               # name: grass/path/rock/flowers/water/castle_wall/desert/snow/lava/swamp/dungeon/void
├── UI/         ui_{name}.png                 # name: icon_coin/icon_heart/btn_tower_archer 等
│               roguelike_{name}.png          # name: atk_up/aspd_up/range_up 等
├── Heroes/     hero_{name}_{form}.png        # name: ranger/knight/witch... form: full/half/avatar
├── VFX/        vfx_{name}.png                # name: explosion/ice_particle/muzzle_flash 等
└── Backgrounds/ bg_{name}.png                # name: main_menu/battle/gacha 等
```

**热替换机制**：所有资源通过 `SpriteLoader` 的缓存+回退策略，放入 `Resources/Sprites/` 对应目录后自动生效，无需修改代码（塔/怪物/投射物/地图已接入，UI/VFX/英雄需要额外代码接入）。
