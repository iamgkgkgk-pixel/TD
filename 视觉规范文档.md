## 二、字体规范

### 2.1 字体选型

| 维度 | 规范 | 说明 |
|------|------|------|
| **字体技术** | BMFont 位图字体（.fnt + .png） | WebGL不支持系统字体动态渲染 |
| **中文字库** | 常用汉字 3000字 + 游戏专用词汇 200字 | 覆盖99.5%场景 |
| **英文/数字** | ASCII全字符集（32-126） | 包含标点符号 |
| **中文字体源** | **思源黑体 Noto Sans SC**（开源免费） | 现代感+可读性+免费商用 |
| **英文/数字字体源** | **Poppins**（Google Fonts，开源免费） | 圆润几何感，与精致卡通风格匹配 |
| **伤害数字字体** | **自定义BMFont**（仅0-9 + K/M/% + ,）| 高辨识度的战斗数字 |

## 三、图标风格规范

### 3.1 图标风格定义

| 维度 | 规范 | 说明 |
|------|------|------|
| **整体风格** | 圆润填充式（Filled Rounded） | 与「精致卡通」风格一致 |
| **描边** | 2px深色描边（`#2C3E50`） | 统一描边增加辨识度 |
| **填充** | 渐变填充（上亮下暗，10%色差） | 微立体感，不纯平 |
| **圆角** | 外轮廓圆角≥4px | 圆润亲和 |
| **视觉重量** | 图标在画布中占比70-80% | 留出呼吸空间 |
| **阴影** | 底部1px `#00000020` | 微弱阴影增加立体 |
| **颜色数量** | 每个图标≤3色（主色+辅色+高光） | 控制复杂度 |
