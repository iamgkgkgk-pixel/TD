## 二、数据公式汇总

### 2.1 基础属性公式

```
怪物HP(章) = 基准HP × 章节倍率
怪物护甲(章) = 基准护甲 × (1 + (章节-1) × 0.08)
怪物魔抗(章) = 基准魔抗 × (1 + (章节-1) × 0.08)
怪物移速(章) = 基准移速 + 章节微调（详见各怪物）
击杀金币(章) = 基准金币 × (1 + (章节-1) × 0.06)

章节倍率 = 1.0 + (章节-1) × 0.12

难度模式叠加：
  困难: HP/护甲/魔抗 × 1.5，移速不变，金币不变
  噩梦: HP/护甲/魔抗 × 2.5，移速不变，金币不变

波次内倍率（CSV中未计入，运行时计算）：
  波次倍率 = 1.0 + (波次序号-1) × 0.05
```

### 2.2 Boss专用公式

```
Boss HP(章) = 基准BossHP × (1 + (章节-首次出现章节) × 0.15)²
  火龙首次出现: 第5章, 基准HP=5000
  石巨人首次出现: 第8章, 基准HP=8000

Boss其他属性: 专属配置（非公式驱动）
```

### 2.3 精英怪公式

```
精英怪属性 = 对应基准 × 章节倍率 × 精英倍率(3.0)
精英怪击杀金币 = 基准 × 章节金币倍率 × 3.0
```

## 五、数值验证

### 5.1 DPS需求验证（关键章节）

```
=== 第1章 普通难度 ===
步兵HP: 200, 每波10只 = 总HP 2,000
箭塔3级DPS: 72.0 → 杀1只步兵需: 200/72 = 2.8秒
每波30秒, 1个3级箭塔可杀: 30/2.8 ≈ 10.7只 ✅ 足够

=== 第10章 普通难度 ===
步兵HP: 416, 每波15只 = 总HP 6,240
需要DPS: 6,240/30 = 208 DPS/波
3个3级箭塔: 216 DPS → 刚好够 ✅ (需要词条加成才能轻松)

=== 第20章 普通难度 ===
步兵HP: 656, 每波20只 = 总HP 13,120

  3个3级塔(~200 DPS) ✅ 可击杀

  需要5+塔 + 词条加成 ✅ 需要好Build

  需要完美Build + 满配阵容 ✅ 终极挑战

```
=== 击杀金币随章节增长 ===

## 六、CSV使用指南

### 6.1 导入Excel

1. 打开Excel → 文件 → 导入 → 选择CSV文件
2. 分隔符选择「逗号」
3. 所有列选择「文本」格式（避免数字格式问题）
4. 导入完成后，对chapter列做筛选，可查看任意章节的全部怪物

### 6.2 困难/噩梦模式计算

CSV中的数据均为**普通难度**。如需困难/噩梦模式数据：

```
困难模式:
  HP = CSV中HP × 1.5
  护甲 = CSV中护甲 × 1.5
  魔抗 = CSV中魔抗 × 1.5
  移速/金币不变

噩梦模式:
  HP = CSV中HP × 2.5
  护甲 = CSV中护甲 × 2.5
  魔抗 = CSV中魔抗 × 2.5
  移速/金币不变
```

### 6.3 波次内倍率计算（运行时）

```
波次内HP = CSV中HP × (1.0 + (波次序号-1) × 0.05)

示例: 第10章 步兵 波次7:
  HP = 416 × (1.0 + 6 × 0.05) = 416 × 1.30 = 541
```

### 6.4 数据筛选建议

| 需求 | Excel筛选方式 |
|------|-------------|
| 查看某章全部怪物 | 筛选chapter=目标章节 |
| 查看某种怪物30章成长 | 筛选enemy_type=目标怪物 |
| 查看所有Boss | 筛选category=Boss |
| 查看精英怪 | 筛选category=Elite |
| 查看首次出现的怪物 | 筛选notes列包含「首次出现」 |
