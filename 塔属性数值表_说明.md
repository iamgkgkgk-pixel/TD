## 二、塔属性数值表字段说明

| `aoe_range` | float | AOE范围半径（格，0=无AOE） |

### 3.3 经济决策矩阵

```
核心决策：260金（1个3级箭塔）vs 280金（2个1级箭塔+1个1级冰塔）

方案A: 1×箭塔3级
  DPS: 72.0 单体 + 穿透等效93.6
  优势: 特殊能力（穿透），集中火力
  
方案B: 2×箭塔1级 + 1×冰塔1级
  DPS: 30.0×2 + 8.0 = 68.0 + 减速20%（等效+47%停留）
  等效DPS: 68.0 × 1.47 = 99.96
  优势: 覆盖面广，有控制
  
结论: 方案B等效DPS略高(100 vs 93.6)，但方案A有穿透质变
→ 两种路线各有优势，平衡健康 ✅
```

## 四、永久升级加成系数说明

### 4.1 设计原则

| 原则 | 说明 |
|------|------|
| **渐进增长** | 攻击倍率从1.00→1.75（+75%），攻速从1.00→1.30（+30%） |
| **攻击为主** | 攻击增长最大（+75%），是养成的核心体感 |
| **攻速为辅** | 攻速增长适中（+30%），避免攻速过快破坏节奏 |
| **射程保守** | 射程增长最小（+12%），过高射程会破坏关卡设计 |
| **金矿适中** | 金矿产出+60%，让经济Build后期更强 |
| **里程碑跳跃** | 第10/15/20/25/30章有明显属性跳跃 |

### 4.2 加成计算公式

```
实际攻击力 = 塔基础攻击力 × tower_permanent_atk_multiplier
实际攻速 = 塔基础攻速 × tower_permanent_atkspd_multiplier
实际射程 = 塔基础射程 × tower_permanent_range_multiplier
实际金矿产出 = 金矿基础产出 × gold_mine_permanent_multiplier

实际DPS = 实际攻击力 × 实际攻速
```

### 4.3 关键节点验证

| 章节 | 攻击倍率 | 攻速倍率 | DPS等效倍率 | 与怪物HP成长对比 | 平衡状态 |
|------|---------|---------|-----------|----------------|---------|
| 1章 | ×1.00 | ×1.00 | ×1.00 | 怪物HP基准 | ✅ 基准 |
| 10章 | ×1.15 | ×1.06 | ×1.22 | 怪物HP×2.08 | 需要更多塔数量补偿 |
| 15章 | ×1.30 | ×1.12 | ×1.46 | 怪物HP×2.68 | 需要3级塔+词条 |
| 20章 | ×1.45 | ×1.18 | ×1.71 | 怪物HP×3.28 | 需要完整Build |
| 25章 | ×1.60 | ×1.24 | ×1.98 | 怪物HP×3.88 | 需要完美Build+养成 |
| 30章 | ×1.75 | ×1.30 | ×2.28 | 怪物HP×4.48 | 需要顶级Build+满养成 |

> **DPS增速(2.28倍) < HP增速(4.48倍)**：这是故意设计！差距通过以下途径弥补：
> 1. 更多塔数量（后期关卡提供更多建造位）
> 2. 词条加成（Roguelike词条可提供100-300%额外DPS）
> 3. 英雄技能（关键时刻爆发）
> 4. 策略优化（更好的塔位选择和目标策略）

## 五、DPS vs 怪物HP平衡验证

### 5.1 验证方法论

```
假设条件：
1. 塔配置 = 当前章节合理的塔数量和等级组合（不含词条加成）
2. 永久升级 = 对应章节的永久升级倍率
3. 有效DPS = 总裸DPS × 攻击倍率 × 攻速倍率
4. 击杀时间 = 怪物有效HP / 有效DPS
5. 有效HP = HP / (1 - 护甲减免比)
   物理减免 = armor / (armor + 200)
   魔法减免 = magic_resist / (magic_resist + 200)
6. DPS盈余率 = (有效DPS × 波次时长 / 全波怪物总HP) - 1
```

### 5.2 关键平衡发现

| 发现 | 详情 | 处理 |
|------|------|------|
| ✅ 前10章DPS充裕 | 盈余率200-500%，教学/入门阶段玩家容错空间大 | 设计预期 |
| ✅ 11-20章逐步收紧 | 盈余率降至250-350%，需要开始注意塔阵搭配 | 设计预期 |
| ⚠️ 15章Boss需37s | 接近45s波次时限，纯裸DPS勉强够 | 需要1-2个攻击词条辅助 |
| ⚠️ 20章Boss需41s | 接近50s波次时限，裸DPS刚好 | 需要词条加成+好的布阵 |
| 🔴 25章Boss需48s | 55s时限内偏紧 | 需要完整Build（3+攻击词条） |
| 🔴 30章Boss需59s | 60s时限内极紧 | 需要完美Build+满养成才能通关 |
| ✅ 普通怪击杀始终<3s | 全章节普通怪击杀在0.7-2.5s | 清怪流畅不拖沓 |
| ✅ 骑士击杀3-5s | 高甲目标需要更多时间但不过分 | 有压力但不卡关 |
