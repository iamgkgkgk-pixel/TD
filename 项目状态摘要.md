## 🏗️ 关键技术架构决策

### Manager初始化顺序（14个Manager）

```
1. EventBus (纯C#)          ← 事件基础设施，最先初始化
2. TimerManager (Mono)       ← 定时器
3. ObjectPoolManager (Mono)  ← 对象池
4. ResourceManager (Mono)    ← 资源加载
5. SaveManager (纯C#)        ← 本地存储
6. AudioManager (Mono)       ← 音频
7. UIManager (Mono)          ← UI框架
8. SceneController (Mono)    ← 场景管理
9. CrashReporter (Mono)      ← 异常上报（尽早捕获）
10. HttpClient (Mono)        ← 网络层
11. WXBridgeExtended (Mono)  ← 微信桥接
12. AnalyticsManager (纯C#)  ← 数据埋点
13. TimeSync (纯C#)          ← 时间同步
14. WXLogin (纯C#)           ← 微信登录
```

### 命名空间组织
- `AetheraSurvivors.Framework` — 核心框架（Singleton/EventBus/ObjectPool/UI/Save/...）
- `AetheraSurvivors.Platform` — 平台适配（WXBridge/HttpClient/Analytics/Login/...）
- `AetheraSurvivors.Editor` — Editor扩展（LevelEditor/BuildTool/GMPanel/...）
- `AetheraSurvivors.Tests` — 测试代码

### 平台适配策略
- 条件编译 `#if UNITY_WEBGL && !UNITY_EDITOR` 区分微信/编辑器
- 编辑器自动Mock所有微信API（无需微信环境即可开发调试）
- 存储抽象层 `IStorageProvider`（EditorStorage / WXStorage）
